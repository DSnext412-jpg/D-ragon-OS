# 🧠 WHAT IS D'RAGON OS?

> D'Ragon OS is **not a Linux distribution** and **not a Windows replacement**.

It is a custom desktop operating system experience built as a **native Windows application** using C++.

The goal is to recreate the feeling of a real operating system while experimenting with operating system architecture, desktop environments, graphics rendering, process management concepts, and modern UI design. It is intended as both a learning project and a showcase of advanced desktop engineering concepts. :contentReference[oaicite:0]{index=0}
